RoxboroughCF is a commercial typeface (by Connary Fagen) and cannot be
bundled here without a license.

If you own it, drop these files into this folder and the site will use
them automatically (no code changes needed):

  RoxboroughCF-Regular.woff2        (or RoxboroughCF-Regular.otf)
  RoxboroughCF-RegularItalic.woff2  (or RoxboroughCF-RegularItalic.otf)

Until then, the site falls back to Fraunces (Google Fonts), a close
free alternative with the same elegant high-contrast serif feel.

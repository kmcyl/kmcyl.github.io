RoxboroughCF.ttf is Kaitlyn's licensed copy of Roxborough CF (by
Connary Fagen) and is used for all serif/display type on the site.
Italic styles are synthesized by the browser from this regular face.

If you ever add more weights or a true italic, drop the files here and
extend the @font-face rules at the top of css/style.css.

Fraunces (Google Fonts) remains as the fallback if this file fails to
load.
